# Haircut Tools Overview & Specifications (Personal Reference)

This document summarizes **all core haircut tools**, their **purpose**, and **common specifications**.
It is designed as a **long-term personal reference** to understand how haircuts are executed and how to communicate clearly with a stylist.

---

## 1. Core Haircut Tools (Overview)

In professional men’s haircuts, there are **three core cutting tools**:

1. **Scissors**
2. **Electric Clippers**
3. **Thinning Scissors**

These three tools perform **nearly all structural haircut work**.

Supporting tools (comb, razor) assist but do not define length or density by themselves.

---

## 2. Scissors (Hair Cutting Scissors)

### Purpose
- Define **overall shape**
- Control **top length**
- Create **natural variation**
- Final detail refinement

### Common Specifications
- Blade length: **5.0 – 6.5 inches**
- Blade type:
  - Straight blade (most common)
- Material:
  - Stainless steel or Japanese steel
- Cutting style:
  - Full cut (cuts all hair contacted)

### Typical Usage Areas
- Top
- Fringe (front)
- Crown
- Fine blending work

### Key Principle
> Scissors control **structure**, not uniformity.

---

## 3. Electric Clippers 

### Purpose
- Cut hair to a **uniform, fixed length**
- Create clean sides and back

### Common Specifications
- Motor type:
  - Rotary or magnetic motor
- Cutting action:
  - Continuous blade movement
- Power:
  - Corded or cordless

### Guard System (Length Control)

| Guard | Length |
|---|---|
| #1 | 3 mm |
| #2 | 6 mm |
| #3 | 10 mm |
| #4 | 13 mm |
| #5 | 16 mm |
| #6 | 19 mm |
| #7 | 22 mm |
| #8 | 25 mm |

### Typical Usage Areas
- Sides
- Back / nape
- Base length definition

### Key Principle
> Clippers define **length uniformity**, not shape.

---

## 4. Thinning Scissors 

### Purpose
- Reduce **hair density**
- Remove internal weight
- Improve movement and softness

### Common Specifications
- Tooth count:
  - 20–40 teeth
- Cut ratio:
  - ~20%–40% hair removed per cut (depends on model)
- Blade structure:
  - One straight blade + one toothed blade

### Typical Usage Areas
- Middle layers of the top
- Internal sections only

### Usage Rules
- 1–2 cuts per area
- Different positions
- Stop early and observe

### Key Principle
> Thinning scissors control **density**, not length.

---

## 5. Supporting Tools (Auxiliary)

### Comb 
- Purpose:
  - Control angle and elevation
  - Assist blending
- Used with:
  - Clippers (clipper-over-comb)
  - Scissors (scissors-over-comb)

### Razor 
- Purpose:
  - Texture effects
  - Edge softening
- Note:
  - Optional
  - Not required for conservative or natural styles

---

## 6. Tool-to-Area Mapping

| Head Area | Tool |
|---|---|
| Sides | Clippers |
| Back / Nape | Clippers + blend |
| Top | Scissors |
| Fringe | Scissors |
| Density control | Thinning scissors |

---

## 7. Final Summary Rule

> Scissors define **shape**.  
> Clippers define **length**.  
> Thinning scissors define **density**.

If unsure during a haircut:
> **Stop early. You can always cut more later.**
