# Number Formatting: Usage of "." and "," in Turkey, China, and the UK

## 1. Overview

Different countries use the period (.) and comma (,) differently when
writing numbers.

The main difference is: - Which symbol is used as the **decimal
separator** - Which symbol is used as the **thousands separator**

------------------------------------------------------------------------

## 2. Turkey

**Format Example:**\
1.234,56

Meaning: - "." = thousands separator\
- "," = decimal separator

Example: - 1.234 = one thousand two hundred thirty-four\
- 1.234,56 = one thousand two hundred thirty-four point fifty-six

This format is common in most European countries.

------------------------------------------------------------------------

## 3. China

**Format Example:**\
1,234.56

Meaning: - "," = thousands separator\
- "." = decimal separator

Example: - 1,234 = one thousand two hundred thirty-four\
- 1,234.56 = one thousand two hundred thirty-four point fifty-six

China follows the same system as the US and UK in modern financial and
technical writing.

------------------------------------------------------------------------

## 6. Important Note

When converting between Turkish and Chinese/UK formats:

Turkish:\
1.234,56

Chinese/UK equivalent:\
1,234.56

If you confuse them, you might misunderstand the value completely.

For example: - 1.234 in Turkey = 1,234\
- 1.234 in UK/China = 1.234

Always check the country format in financial documents.
